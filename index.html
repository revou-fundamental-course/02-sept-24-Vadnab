Luas Segitiga

Rumus Luas Segitiga yaitu:

L = 1/2 x a x t

Di mana:
L = Luas
a = Panjang Alas
t = Tinggi

Hitung Luas Segitiga




Keliling Segitiga

Rumus Keliling Segitiga yaitu:

K = s1 + s2 + s3

Di mana:
s = sisi

Luas Jajar Genjang

Rumus Luas Jajar Genjang yaitu:

L = a x t

Di mana:
L = Luas
a = Panjang Alas
t = Tinggi

Hitung Luas Jajar Genjang




Keliling Jajar Genjang

Rumus Keliling Jajar Genjang yaitu:

K = a + b + c + d

Di mana:
a dan c = alas/tutup
b dan d = sisi miring
